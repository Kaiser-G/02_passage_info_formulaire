Exercice 3

Ecrire une page qui affiche un formulaire permettant de saisir un LOGIN et un MOT DE PASSE.
Lorsque l’utilisateur clique sur le bouton envoyer cela envoie les informations vers la page
Passage_info01_recup_param.html (cf. exercice 1) qui les affichent.
Remarque : Le bouton ENVOYER ne sera pas codé à l’intérieur du formulaire (pas un submit).
Il faudra donc coder du Javascript pour intercepter le CLICK sur le bouton puis générer la redirection vers la deuxième page en transmettant les paramètres avec les valeurs.
Url à construire : http://... Passage_info01_recup_param.html?LOGIN=’Licois’&PASSWORD=’Johanne’
Modifier le window.location.href = la nouvelle URL
